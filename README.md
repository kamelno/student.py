# Student Management System

A student management system built with Python OOP for managing student records and grades.

## Features

- ✓ Add and delete students
- ✓ Record grades for students
- ✓ Calculate grade averages
- ✓ View all students and their averages
- ✓ View individual student information

## Requirements

- Python 3.8+

## Installation

```bash
# Clone the repository
git clone https://github.com/YOUR_USERNAME/student-management-system.git
cd student-management-system

# Run the program
python student.py
```

## Usage

```bash
python student.py
```

Use the interactive menu to manage students and grades.
